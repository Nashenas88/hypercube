# Hypercube

A 4D Rubik's cube visualization built with Rust, iced, and wgpu.

## Features

- Interactive 4D hypercube rendering
- GPU-accelerated graphics with wgpu
- Adjustable sticker and face scaling
- Real-time 4D to 3D projection

## Usage

```bash
cargo run
```
